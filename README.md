# Galaxy

A H5 game to test reaction ~~（The name was taken in vain）~~.

## Project setup

```
yarn
// or
yarn install
```

### Compiles and hot-reloads for development

```
yarn start
```

### Compiles and minifies for production

```
yarn build
```

### Preview production build products locally

```
yarn build
// and then
yarn serve
```
