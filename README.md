# Galaxy

考验反应力的 H5 小游戏 ~~（名字是瞎取的）~~。

## Project setup

```
yarn
// or
yarn install
```

### Compiles and hot-reloads for development

```
yarn start
```

### Compiles and minifies for production

```
yarn build
```

### Preview production build products locally

```
yarn build
// and then
yarn serve
```
